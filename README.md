Shogi App - Multiplayer Local Edition
​本格的な将棋をブラウザで楽しめるWebアプリケーションです。
1台のデバイスを2人で共有して対局する「ローカル対戦」に特化しています。
​特徴
​振り駒機能: 対局開始時に自動で先攻・後手（先手・後手）を決定します。
​盤面回転: 友達と向き合ってプレイしやすいよう、盤面を180度回転させる機能があります。
​本格デザイン: 五角形の駒形状、成り駒の赤字表示、榧（かえ）の盤をイメージしたデザイン。
​レスポンシブ: PC、タブレット、スマートフォンのブラウザで動作します。
​使い方
​index.html をブラウザで開きます。
​起動時の「振り駒」演出で先手・後手が決まります。
​「対局開始」ボタンを押してスタートします。
​駒をタップして移動先を選択します。成れる場合は選択肢が表示されます。
